1. Structural

Here I talk about The primaries tags on HTML that make the structure of the archive. They are the tags: html, head and body.


1.1 <html> </html>

the html tag demark all the composition of the page. It's inside this tag that will contain all the code used to make the web page.


1.2 <head> </head>

This tag encapsulate all thing that are in your page but isn't show to the user, such as css code, formatting settings and other things.


1.3 <body></body>

In this tag we put everything that goes in the page itself. Every text, image and every other thing that the user will interact.


2. Formatting

Now I will talk about the tags that format the text and are the focus of this exercise. They are: <meta charset="UTF-8" /> , <br /> <p></p>, <ul></ul> and <li></li>


2.1 meta charset = "UTF-8"

The meta tag is a element used to define metadata, in this tag we can put attributes as the charset that can't go in others tags. The charset attribute is used to identify the codification of the characters in the file, the more basic one is ASCII, the UTF-8 utilized in this document is used to make possible the programming in documents of any language, being commonly used in web applications.


2.2 br /

The <br /> tag is simple used to make a vertical space in the text.


2.3 <p></p>

the <p></p> tag indicates a paragraph. Everything put inside it is encapsulate in the same paragraph.


2.4 <ul></ul> and <li></li>

the tags <ul></ul> and <li></li> go together to list items, as the list of tags at the start of the page, the <ul></ul> marks the place where all the items will be put while <li></li> marks each item.
