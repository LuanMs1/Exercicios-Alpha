In the next sessions I will make a brief explanation on each of the tags used to complete the exercise. I will subdivede in thre categories: Extrutural, Text and formating.


1. Extrutural

Here I talk about The primaries tags on HTML that make the Extrutucture of the archive. They are the tags: html, head and body.


1.1 <html> </html>

the html tag demark all the composition of the page. It's inside this tag that will contain all the code used to make the web page.


1.2 <head> </head>

This tag encapsulate all thing that are in your page but isn't show to the user, such as css code, formating settings and other things.


1.3 <body></body>

In this tag we put everything that goes in the page itself. Every text, image and every other thing that the user will interact.
